# 1. PRD: 의약품 대체 탐색 기반 통합검색 개선

> 작성일: 2026-03-13 | 작성자: 장미나 | 버전: v1.0

---

## 1. 배경 및 목적

의료기관 종합몰에서 의약품 검색은 주문 업무의 핵심 기능.

의사·의료기관 담당자는 특정 상품명보다 **치료 목적, 효능, 투여 방식, 성분**을 기준으로 대체 의약품을 탐색하는 경우가 많다. 그러나 현재 검색은 상품명 중심으로 동작하여 이러한 탐색 목적을 지원하지 못함.

| 사용자 검색 | 현재 결과 | 기대 결과 |
|------------|-----------|-----------|
| 해열 | 상품명에 '해열' 포함된 것만 노출 | 해열 효과 의약품 목록 |
| 근육주사 | 상품명 매칭 없음 | 근육주사 투여 의약품 목록 |
| 아세트아미노펜 | 성분명 검색 안 됨 | 동일 성분 대체 의약품 목록 |

**개선 목적:** 치료 목적·효능·성분·투여 방식 기반으로 대체 의약품을 탐색할 수 있도록 검색 기능을 개선.

## 2. 목표 및 성공 지표

| 목표 | 성공 지표 |
|------|----------|
| 대체 의약품 탐색 지원 | 의약품 정보 기반 검색 결과 클릭률(CTR) |
| 검색 정확도 개선 | 정확 일치 상품 상단 노출 비율 |
| 재구매 편의성 강화 | 검색 → 주문 전환율 |

---

## 3. 검색 기능 구조

검색 기능은 세 가지 유형으로 동작한다.

| 검색 유형 | 검색 대상 |
|-----------|-----------|
| 상품 기반 | 상품명, 카테고리(소분류), 규격, 제조사, 공급사, 보험코드 |
| 의약품 정보 기반 | 성분명, 효능·효과 (`pharmaceutical_effects_ko`) |
| 사용자 기반 | 구매 이력 |

> 의약품 정보는 원스글로벌 데이터를 활용한다.

## 4. 핵심 기능

### 4.1 의약품 정보 기반 검색

검색어를 의약품 효능·성분 데이터와 매칭하여 대체 의약품 후보를 제공한다.

**동작 흐름:** 검색어 입력 → 의약품 정보 매칭 → 연결 상품 조회 → 결과 노출

### 4.2 검색 결과 우선순위

| 순위 | 기준 |
|------|------|
| 1순위 | 상품명 완전 일치 (공백 무시) |
| 2순위 | 상품명 부분 포함 |
| 3순위 | 의약품 정보 매칭 (효능·성분) |
| 4순위 | 상품 부가정보 매칭 (제조사 > 규격 > 단위 > 특이사항 > 공급사) |

### 4.3 검색 결과 정렬

기본 정렬: **판매순** / 옵션: 신상품순, 낮은가격순

### 4.4 검색어 자동완성

상품명 우선, 인기 검색어 반영 / 응답속도 0.5초 이내

### 4.5 연관 검색어

최소 1개 ~ 최대 7개 노출

| 우선순위 | 출처 | 노출 수 |
|----------|------|---------|
| 1순위 | AX 개발팀 키워드 (진료과 필터 적용) | 최대 3개 |
| 2순위 | 관리자 등록 키워드 | 나머지 채움 |

클릭 시 해당 키워드로 재검색 / 관리 위치: `관리자 > 상품관리 > 상품 수정 > 연관검색어`

### 4.6 내 주문 상품 우선 노출

현재 검색 결과 포함 + 구매 이력 존재 시 상단 별도 영역 노출 (최대 3개, 초과 시 슬라이더)

```
[내가 구매했던 상품]
─────────────────────
[일반 검색 결과]
```

### 4.7 이벤트 / 기획전 노출

검색 상품·키워드·공급사와 연관된 이벤트 존재 시 결과 내 배너 1개 노출

| 상품 수 | 노출 위치 |
|---------|-----------|
| 3개 미만 | 미노출 |
| 3~7개 | 첫 번째 줄 우측 |
| 8개 이상 | 두 번째 줄 우측 |

### 4.8 페이징

페이지당 최대 80 슬롯 (이벤트 배너 포함 시 상품 79 + 이벤트 1) / 이전·다음·처음·마지막

## 5. 비기능 요구사항 및 예외 처리

| 항목 | 기준 |
|------|------|
| 검색 응답속도 | 2초 이내 |
| 자동완성 응답 | 0.5초 이내 |
| 지원 환경 | PC |
| 검색 결과 없음 | "검색 결과 없음" 안내 메시지 표시 |
| 구매·이벤트 이력 없음 | 해당 영역 미노출 |
