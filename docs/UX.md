## 1. UX 목적

현재 미소몰 검색은 상품명 기반으로 동작하지만, 실제 의료기관 사용자는 **증상·용도 기반으로 의약품을 탐색**하는 경우가 많음. 자연어 검색을 통해 이 탐색 경험을 지원.

**검색 예시:** `소아 해열 주사` · `케토신주 대체약` · `항생제 주사`

---

## 2. 사용자 흐름

```
Search Input → AI Query Analysis → Product Recommendation → Product Detail
```

| 단계 | 사용자 | 시스템 |
|------|--------|--------|
| Search Input | 자연어로 검색어 입력 | 자동완성 제공 |
| AI Query Analysis | 결과 대기 | 증상·약품명·약물군 분석 |
| Product Recommendation | 추천 결과 확인 | 대체 의약품 추천 리스트 노출 |
| Product Detail | 상품 선택 | 상품 상세 페이지 이동 |

**추천 예시**
```
검색어: 소아 해열 주사
추천 결과: 케토신주 / 이부프로펜주 / 덱시부프로펜주
```

---

## 3. 주요 UI 구성

- **검색창** — 자연어 입력 + 성분·효능 기반 자동완성
- **AI 분석 상태** — 검색어 분석 진행 표시
- **추천 결과** — 내 구매이력 상품 · 연관검색어 · 의약품 리스트 · 대체의약품 버튼
- **상품 상세** — 동일 성분·효능 대체 의약품 목록
