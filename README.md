# 미소몰 의약품 대체 탐색 검색 개선
의료기관 종합몰(미소몰)의 의약품 검색 정확도를 개선하는 서비스 기획 프로젝트.
상품명 중심 검색에서 **효능·성분·투여 방식 기반 대체 의약품 탐색**으로 전환하는 것을 목표로 함.

---

## 배경
의사·의료기관 담당자는 특정 상품명보다 치료 목적 기반으로 의약품을 탐색하는 경우가 많음.

| 상황 | 검색 예시 |
|------|-----------|
| 동일 효능 대체 의약품 탐색 | `해열`, `진통제` |
| 투여 방식 기준 탐색 | `근육주사` |
| 동일 성분 대체 의약품 탐색 | `아세트아미노펜` |

현재 검색은 상품명 매칭 중심으로 동작하여 이러한 탐색 흐름을 지원하지 못함.

---

## 프로젝트 범위

**포함**
- 의약품 정보 기반 검색 로직 설계
- 검색 결과 우선순위 정렬 정의
- 검색 결과 UI 구조 설계

**제외**
- 원스글로벌 API 신규 개발 (기존 연동 활용)
- 관리자 기능 개발
- 모바일 환경 지원

## 기술 스택

| 구분 | 기술 |
|------|------|
| 프론트엔드 | JavaScript, Vue 3 (SPA) |
| 백엔드 | ASP.NET MVC / .NET Core |
| DB | RDBMS (관계형 데이터베이스 기반 상품·의약품 데이터 관리) |
| 외부 API | 원스글로벌 API (의약품 성분·효능 데이터 연동) |

---

## 데이터 구조

| 구분 | 항목 |
|------|------|
| 상품 데이터 | 상품명, 카테고리, 제조사, 규격, 공급사, 보험코드 |
| 의약품 데이터 | 성분명, 효능·효과 (`pharmaceutical_effects_ko`) |
| 데이터 출처 | 원스글로벌 API 의약품 데이터 |

---

## 대상 사용자
- 의사 — 치료 목적 기반 의약품 탐색
- 간호사 — 투여 방식 기반 의약품 탐색
- 의료기관 주문 담당자 — 대체 의약품 주문
